# T009 — Responsive web app (mobile + desktop)

| Field | Value |
|-------|--------|
| **Status** | todo |
| **Priority** | high |
| **PRD** | §3.1, §5.2 (Responsive web app; web only) |

## Description

Ensure the app is responsive and usable on both mobile and desktop. Layout, navigation, stat toggles, time scope filter, and rivalries list (including casual/hardcore views) must work and remain readable across viewport sizes.

## Acceptance criteria

- [ ] Core flows (view rivalries, change stat toggles, change time scope, drill into stats) work on mobile and desktop
- [ ] Touch targets and spacing are adequate on small screens
- [ ] No horizontal scroll or clipped content for typical viewports
- [ ] Optional: define breakpoints and document layout behavior

## Notes

- MVP is web only; no native app or PWA (PRD §3.2, §5.2).
