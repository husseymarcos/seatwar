## Description

Provide a **casual** entry point where each rivalry shows one “overall” winner (e.g. one driver ahead on balance). This can be based on the selected stat(s) and time scope — e.g. primary stat, or a weighted combination — and must stay consistent with the “who’s winning” logic.

## Acceptance criteria

- [ ] Each rivalry displays a single overall winner for the current stat selection and time scope
- [ ] Definition of “overall” is documented (e.g. primary stat, or weighted combo)
- [ ] Casual view is the default or clearly accessible (e.g. main list/cards)
- [ ] Works with stat toggles and time scope filter
