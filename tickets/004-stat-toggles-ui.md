# T004 — Stat toggles (points, race finishes, qualifying)

| Field | Value |
|-------|--------|
| **Status** | todo |
| **Priority** | high |
| **PRD** | §3.1 (Stat toggles: user can choose which stat(s) drive the view) |

## Description

Add UI controls so the user can choose which stat(s) drive the view: **Points**, **Race finishes**, **Qualifying**. The main view (rivalries list / cards) should update to reflect the selected stat(s).

## Acceptance criteria

- [ ] User can select one or more of: Points, Race finishes, Qualifying
- [ ] Selected stat(s) drive what “who’s winning” is based on (and what is displayed)
- [ ] Toggle state is clear and works on mobile and desktop
- [ ] Default selection is documented (e.g. Points only or all three)
